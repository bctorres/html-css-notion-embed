Latent Dirichlet allocation
A generative statistical model in natural language processing
Definition
A Bayesian network and generative statistical model used for modeling automatically extracted topics in textual corpora.
Domain
Natural language processing
Function
Observations (e.g., words) are collected into documents, with each word's presence attributable to one of the document's topics.
Topics handling
Each document contains a small number of topics, facilitating topic analysis and content categorization.
